# Personal Notes

Target audience:
- Recruiters
- Hiring managers
- Technical interviewers

Tone:
- Elegant
- Premium
- Modern
- Professional
- Confident, not flashy

Design preferences:
- Smooth animations
- Luxury/minimal aesthetic
- Strong typography
- Dark mode default
- Mobile-first responsive

Must-have sections:
- Hero
- About
- Experience
- Projects
- Skills
- Certifications / Education
- Contact

Nice-to-have:
- Resume download button
- GitHub link
- Subtle scroll animations
- Project cards with hover interactions

Important constraints:
- Do not invent any skills, metrics, employers, dates, certifications, or project results.
- If anything is missing, create a TODO note instead of guessing.
